# Icônes PWA

Ce dossier doit contenir les icônes pour la Progressive Web App.

## Générer les icônes

Vous pouvez utiliser ces outils en ligne gratuits :

1. **RealFaviconGenerator**
   - https://realfavicongenerator.net/
   - Upload votre logo/image
   - Télécharger le package
   - Copier les icônes PNG dans ce dossier

2. **PWA Builder Image Generator**
   - https://www.pwabuilder.com/imageGenerator
   - Upload votre logo/image
   - Télécharger les icônes
   - Copier dans ce dossier

## Tailles requises

Les icônes suivantes sont référencées dans le manifest.json :

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Recommandations

- Format : PNG avec fond transparent ou couleur unie
- Design : Simple, lisible en petite taille
- Couleur : Cohérente avec le thème de l'application (bleu #2563eb)
